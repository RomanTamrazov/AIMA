## 🎯 О проекте

AI-помощник по медиа — это комплексная система, разработанная для **Центра исследований и разработки Сбера** в Санкт-Петербурге. Платформа использует искусственный интеллект для автоматизации процессов поиска, анализа и участия сотрудников в релевантных IT-мероприятиях города.

## 🚀 Быстрый старт

### Предварительные требования

- Python 3.8 или выше
- Telegram Bot Token
- Доступ к интернету для парсинга

### Установка за 5 минут

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/sber-ai/media-assistant.git
cd AI-Media-Assistant

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Настройте конфигурацию
cp config.example.py config.py
# Отредактируйте config.py - добавьте токен бота

# 4. Запустите систему
python main.py
```
Настройка бота Telegram

Создайте бота через @BotFather
Получите токен и добавьте в config.py:
python
BOT_CONFIG = {
    "token": "YOUR_BOT_TOKEN_HERE",
    "admin_password": "admin123",
    "manager_password": "manager123"
}
Запустите бота и начните работу с команды /start
